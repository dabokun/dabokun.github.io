These astrophotographs have GainMap embedded in them and can be viewed in HDR using a compatible web browser and display. If your display is not compatible, they will be displayed as normal SDR images.
For example, it is likely to be visible on relatively new HDR-compatible smartphones and laptops. It is recommended to use Chrome or Microsoft Edge as your browser. Don't forget to set HDR in your OS.
